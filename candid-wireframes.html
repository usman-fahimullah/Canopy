Design Principles

Universal patterns to apply across all Candid pages

The Core Problem
Everything floats in a white void. There's no visual containment, no depth hierarchy, and no clear zones.

1. Page Structure

Every page needs three zones:
Header ‚Äî Title + actions with bottom border
Toolbar ‚Äî Filters/search in contained bar
Content ‚Äî Main content in a card

2. Warm Empty States

Replace cold gray icons with:
‚Ä¢ Gradient background (green ‚Üí blue)
‚Ä¢ Larger icon size (64-72px container)
‚Ä¢ Friendlier, helpful copy
‚Ä¢ Clear action button

3. Visual Containment

‚Ä¢ Toolbars get border + background
‚Ä¢ Progress sections get cards
‚Ä¢ Hero areas get gradient backgrounds

4. Header Separation

‚Ä¢ Bottom border or background

5. No Inline Errors

‚Ä¢ Toast notifications for transient errors
‚Ä¢ Warm error states for persistent issues
